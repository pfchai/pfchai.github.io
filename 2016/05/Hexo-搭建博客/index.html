Hexo 搭建博客

May 12, 2016 | Hits

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

问题1. 执行hexo deploy后，出现ERROR Deployer not found: git

解决办法，在当前目录下执行sudo npm install hexo-deployer-git --save即可。
